# Disaster Heroes 3D

A 3D educational disaster simulation experience designed for children aged 6–12, developed for the Bursa İnegöl Municipality competition.

The project introduces basic earthquake awareness, safety behavior, and environmental interaction through an interactive Unity-based gameplay system.

Built in Unity with a focus on clarity, accessibility, and structured gameplay systems suitable for early-age learning.

---

🎮 Core Experience Loop
- Explore a simple 3D environment designed for learning
- Experience controlled earthquake simulation events
- Identify safe and unsafe behaviors during disaster scenarios
- Follow guided interactions to reach safety zones
- Learn through repetition, feedback, and visual cues

---

🧠 Core Gameplay Systems

🏗️ Disaster Simulation System
- Controlled earthquake event triggering system
- Physics-based environment reaction (lightweight and optimized)
- Predefined safe zone logic for learning objectives
- Scripted event sequences for educational flow

🚶 Player Guidance System
- Simple movement mechanics optimized for children
- Visual indicators for safe direction guidance
- Interaction prompts for learning reinforcement
- Reduced complexity input system (accessible design)

📚 Educational Feedback System
- Real-time feedback on player decisions
- Positive reinforcement for correct actions
- Visual and audio cues for learning retention
- Scenario-based learning progression

---

⚙️ Engineering & Architecture

🧩 System Design
- Modular gameplay systems (Simulation / Player / UI separation)
- Event-driven architecture for disaster sequences
- ScriptableObject-based configuration for scenarios
- Scalable structure for additional disaster types

🧠 Simulation Control Layer
- Centralized disaster event manager
- Time-controlled sequence execution
- Safe-state validation system
- Reset and replay loop architecture

📦 Optimization Strategy
- Lightweight physics usage for performance stability
- Minimal runtime instantiation (preloaded environment)
- Efficient update loop management
- Stable frame pacing for mobile/low-end devices

---

🛠️ Tech Stack
- Engine: Unity (3D)
- Language: C#
- Architecture: OOP + Event-driven + Modular Design
- Target Platform: PC
